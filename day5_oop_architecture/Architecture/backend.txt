# folder Structure 
src/
├── controllers/      # Presentation Layer: Validates HTTP inputs, returns responses
├── services/         # Business Logic Layer: Core rules, calculations, orchestrations
├── repositories/     # Data Access Layer: Direct database queries (SQL/ORMs)
├── models/           # Data Structures/Schemas (DB tables)
├── config/           # Environment variables, database connections
└── middlewares/      # Auth guards, logging, error handling
